                          ğŸ”§ TROUBLESHOOTING
================================================================================

Problem: Server won't start
Solution: 
- Check if port 3001 is available
- Run: npm install (in server folder)
- Check .env file exists in project root

Problem: Payment not working
Solution:
- Verify both servers are running
- Check browser console for errors
- Ensure you're logged in
- Verify Stripe keys in .env

Problem: Not redirecting after payment
Solution:
- Check VITE_API_URL in .env
- Verify frontend is on port 5173
- Check server logs for errors

Problem: Order not appearing
Solution:
- Check MongoDB connection
- Verify product has stock
- Check server console logs
- Ensure payment completed successfully

When you're ready to go live:

4. Setup webhooks (recommended):
   - Add endpoint: https://yourdomain.com/api/stripe/webhook
   - Get webhook secret
   - Add to .env: STRIPE_WEBHOOK_SECRET=whsec_...

5. Deploy your application

6. Monitor payments in live dashboard
